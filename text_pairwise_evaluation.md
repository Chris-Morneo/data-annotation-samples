# Text Pairwise Evaluation Examples

## Prompt
"What is two-factor authentication?"

### Response A
'Two-factor authentication is a security method that requires a password and a second
form of verification, such as a code sent to your phone.'

### Response B
'Two-factor authentication is when you log in twice using the same password.'

### Selected Better Response
Response A

### Reasoning
Response A correctly explains two-factor authentication and provides a clear example.
Response B is inaccurate and misleading.
